You need a commercial license if you want to offer Hive as part of an

## When Do You NOT Need a Commercial License?

The following uses are free under the BSL 1.1 and do not require a commercial
license:

| Use Case | Commercial License Required? |
|---|---|
| Self-hosting Hive for your own organization | No |
| Using Hive internally, regardless of team size | No |
| Development, testing, staging, and CI/CD | No |
| Building solutions for clients (each client self-hosts) | No |
| Contributing to Hive (bug fixes, features, docs) | No |
| Forking Hive for personal or internal use | No |
| Academic or research use | No |
| Offering Hive as a hosted service to third parties | **Yes** |
